1. 添加多个div到文档中优化

源代码

for( var i=0; i<5; i++){ $("<div>>").appendTo(document.body) }

优化

var s = "<div></div>>"; for( var i=0; i<5; i++){ s += s; } $(s).appendTo(document.body)
